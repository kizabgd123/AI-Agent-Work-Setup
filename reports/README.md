# Reports Directory

This folder contains task batch reports, weekly aggregates, and benchmark comparison reports.

## Naming Convention

| Report Type              | Filename Format                      |
|:-------------------------|:-------------------------------------|
| Weekly summary           | `weekly-summary-YYYY-MM-DD.md`        |
| Pilot batch report       | `pilot-batch-report.md`              |
| Pre/Post comparison      | `prepost-comparison-YYYY-MM-DD.md`   |
| Custom analysis          | `analysis-[topic]-YYYY-MM-DD.md`     |

## Instructions
- Every weekly review concludes with a report saved here
- Reports are not editable after the week closes — add a new file for corrections
