# Tech Stack: AI Agent Work Setup & Coordination Protocol

## Core Technologies
- **Language:** Python (3.8 - 3.12)
- **IDE Environment:** Antigravity IDE
- **Coordination Framework:** Conductor (Spec-driven development)

## AI Agents
- **Gemini CLI:** Project Manager & Lead Architect (Primary decision maker).
- **Qwen Code CLI:** Core Developer (Implementation specialist).
- **Aider:** Editor & Git Master (Refactoring and version control).
- **Open Interpreter:** System Executor & Tester (Runtime verification).

## Infrastructure & Tools
- **Environment Management:** Python Virtual Environments (`.venv`)
- **Version Control:** Git
- **Configuration:** `.env` for API keys, `.aider.conf.yml` for tool-specific settings.
- **Automation Scripts:** Bash and Python wrappers for agent orchestration.

## Documentation & Logging
- **Markdown:** All coordination and tracking files.
- **Logging:** Centralized `Work_log.md`.
